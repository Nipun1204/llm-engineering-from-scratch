# LLM Engineering From Scratch

A hands-on study of language model internals, training, inference,
and serving.

The goal of this repository is to implement core components from
first principles before studying production implementations.

## Progress

- [ ] Language model objective
- [ ] Tokenization
- [ ] Embeddings and positional encoding
- [ ] Attention
- [ ] Transformer
- [ ] MiniGPT
- [ ] Inference optimization
- [ ] Post-training
- [ ] Serving and evaluation
